# Reverse 命令

`reverse` 命令用于查找“哪些端口依赖目标包”。

## 命令语法

```shell
celer reverse <name@version> [flags]
```

## 重要行为

- 必须且只能提供一个包参数。
- 包参数必须为 `name@version`。
- 默认仅检查运行时依赖关系。
- `--dev` 会把开发依赖也纳入反向查找。

## 命令选项

| 选项  | 简写 | 类型 | 说明             |
|-------|------|------|------------------|
| --dev | -d   | 布尔 | 包含开发依赖查找 |

## 常用示例

```shell
# 查询运行时反向依赖
celer reverse eigen@3.4.0

# 查询时包含开发依赖
celer reverse nasm@2.16.03 --dev
```

## 参数校验规则

- 空参数会被拒绝。
- 缺少 `@` 会被拒绝。
- 格式需匹配 `^[a-zA-Z0-9_-]+@[a-zA-Z0-9._-]+$`。
